    MVP 1

    [EP02] Adicionar ou remover produtos

    User Stories
    [US03] Eu, como gerente, quero ser capaz de adicionar um novo produto ao catálogo.
    [US04] Eu, como gerente, quero ser capaz de remover um produto do catálogo.

    [EP01] Consulta de informações de produto

    User Stories
    [US01] Eu, como usuário, quero poder consultar o preço do produto.
    [US02] Eu, como usuário, quero poder consultar o nível de estoque dos produtos agilizar.

    [EP03] Controle de Estoque

    User Stories
    [US08] Eu, como gerente, gostaria do sistema gerar um alerta de estoque baixo, para poder repor o produto.
    [US09] Eu, como gerente, gostaria de impedir a venda de uma quantidade de produtos acima do que existe no estoque.
    [US10] Eu, como gerente, quero ser capaz de visualizar o estoque total da loja em tempo real.

    [EP04] Frente de caixa

    User Stories
    [US11] Eu, como funcionário quero ser capaz de criar um carrinho que contém um pedido de venda que retire o produto automaticamente do estoque.
    [US12] Eu, como funcionário quero adicionar ao carrinho do pedido em aberto, diferentes produtos, utilizando código reservado a cada produto.
    [US13] Eu, como funcionário quero que o carrinho do pedido em aberto some o total de valor de produtos e mostre o valor total da venda.
    [US14] Eu, como funcionário quero que o carrinho tenha um fechamento de venda, e que nele indique o troco a receber.
    [US15] Eu, como funcionário quero ser capaz de escolher a forma de pagamento que foi realizada.
    [US16] Eu, como funcionário quero ser capaz de escolher como foi realizado o pedido (delivery,aplicativo,presencial).
    [US17] Eu, como funcionário, quero ser capaz de registrar a venda através dos códigos de barra.

    MVP 2

    [EP02] Adicionar ou remover produtos

    User Stories
    [US05] Eu, como funcionário, quero ser capaz de registrar produtos através dos códigos de barra para agilizar o processo de atualização do produto.
    [US06] Eu, como gerente, gostaria de criar promoções, para estimular as vendas.

    [EP03] Controle de Estoque

    User Stories
    [US07] Eu, como gerente, quero ser capaz de ver o histórico de alterações no estoque de um produto.

    [EP05] Geração de relatórios de vendas

    User Stories
    [US18] Eu, como gerente, quero ser capaz de gerar um gráfico de linhas, para observar melhor o fluxo de vendas.
    [US19] Eu, como gerente, quero ser capaz de gerar um gráfico de pizza, para identificar a participação de cada.
    [US20] Eu, como gerente, quero ser capaz de gerar um gráfico de barras, para comparar as vendas de diferentes produtos.
    [US21] Eu, como gerente, quero ter um dashboard para acompanhar o desempenho das vendas e a situação financeira da minha loja.
    [US22] Eu, como gerente, quero ser capaz de gerar relatórios de vendas por horário para identificar os horários de pico de vendas.
    [US23] Eu, como gerente, quero ser capaz de gerar relatórios comparando a venda entre diferentes métodos como aplicativos (iFood, Uber Eats), presencial ou delivery(telefone) para identificar a eficácia de cada método.
    [US24] Eu, como gerente, quero ser capaz de gerar uma lista com todas as vendas já realizadas.
    [US25] Eu, como gerente, quero ser capaz de gerar esses relatórios de vendas por período, para monitorar o desempenho.
    [US26] Eu, como gerente, quero ser capaz de gerar relatórios de vendas por promoção para identificar as promoções que têm melhor desempenho.

    [EP06] Gerenciamento de funcionários

    User Stories
    [US27] Eu, como gerente, quero ser capaz de adicionar um novo funcionário ao sistema.
    [US28] Eu, como gerente, quero ser capaz de remover um funcionário do sistema.
    [US29] Eu, como gerente, quero ser capaz de editar as informações de um funcionário existente.
    [US30] Eu, como gerente, quero ser capaz de visualizar as informações pessoais dos funcionários, como nome, número de telefone e endereço.
